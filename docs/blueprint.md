# **App Name**: CV Match Pro

## Core Features:

- User Authentication: Secure user registration, login, and profile management.
- Profile Dashboard: A user dashboard to create, view, and manage job position templates ('cards').
- Template (Card) Creation: Allow users to create and save templates for job position descriptions.
- CV Upload: Enable users to upload multiple CVs (PDF or Word).
- AI-Powered CV Analysis: Leverage AI to analyze uploaded CVs against the selected job position template and provide a percentage match score for each CV. Use an LLM as a tool to determine which data points in a resume will influence matching.
- Results Display: Display a ranked list of candidates with their CVs and percentage match scores.

## Style Guidelines:

- Primary color: Sky blue (#63B5FF) to evoke trust and competence.
- Background color: Light gray (#F2F4F7) for a clean, professional look.
- Accent color: Teal (#47A7FF) for calls to action and important information.
- Body and headline font: 'Inter', a sans-serif font, will be used for the whole page to ensure legibility.
- Use professional, clear icons for navigation and actions.
- A clean and structured layout to provide a good UX.
- Subtle transitions and animations to enhance user experience during analysis and result display.
